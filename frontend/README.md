## Json Server Mock Backend

Para usar um backend mockado

### Json Server usando `yarn`

```bash
yarn run json-server -w -p 3333 ./mock/database.json
```

### Json Server usando `npm`

```bash
npm run json-server -w -p 3333 ./mock/database.json
```
